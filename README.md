# SkyNova Airline Booking - React Project

## Project Idea
SkyNova Airline Booking is a React website for browsing flights, selecting a flight, entering passenger information, choosing extras, and viewing booking summary.

## Requirements Covered
- React project with clear folder structure
- Reusable components
- External CSS
- Internal styling
- Styled Components
- Bootstrap installed and used
- Props between components
- Mapping lists with reusable cards
- Event handling: click, input, form submit, checkbox
- useState
- useEffect
- Forms
- Routing
- API integration using JSONPlaceholder
- Context API for global booking state
- Clean readable code

## How to Run
```bash
npm install
npm run dev
```

Then open the local URL shown in the terminal.
